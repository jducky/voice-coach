# Tabata Voice Coach

Initial scaffold for a Tabata workout app centered on configurable voice guidance.

## Scope of this scaffold

- Product and architecture notes for the MVP
- TypeScript domain models
- Pure session engine for time-based interval execution
- Phase builder for converting routines into a timeline
- Unit tests for the engine

## Planned app stack

- React Native
- TypeScript
- Zustand
- React Navigation
- MMKV
- react-native-tts

## Local commands

```bash
npm install
npm test
```
